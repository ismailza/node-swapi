Node SWAPI

A simple Node.js web application that uses the SWAPI (Star Wars API) to display films and people. Containerized with Docker for easy deployment and scalability.

It's built with Node.js for the backend, Express for server-side logic, Axios for API requests, along with Bootstrap, Vanilla JS, and Font Awesome for the front-end. Docker simplifies setup and ensures consistency across environments.
